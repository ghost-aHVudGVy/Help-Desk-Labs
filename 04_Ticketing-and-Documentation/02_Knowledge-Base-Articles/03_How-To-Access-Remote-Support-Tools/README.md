# Knowledge Base Article: How to Access Remote Support Tools

**Category:** Remote Support & Troubleshooting  
**Audience:** All employees

## Overview

This article provides instructions for connecting to your workstation remotely using approved support tools. It covers **RDP (Remote Desktop Protocol)**, **Quick Assist**, and **TeamViewer**, ensuring secure and reliable remote troubleshooting.

---

## Using Remote Desktop (RDP)

1. Ensure your computer is connected to the **corporate network** or VPN.  
2. Press **Win + R**, type `mstsc`, and press **Enter**.  
3. In the **Computer** field, enter the **hostname or IP address** of the remote computer.  
4. Click **Connect**.  
5. Enter your **domain credentials** when prompted.  
6. Click **Yes** if asked to **verify the remote certificate**.  
7. Once connected, you can control the remote computer as if you were physically at the workstation.

---

## Using Quick Assist

1. Open **Quick Assist** from the Start menu.  
2. Choose **Assist another person** (if you are IT) or **Get assistance** (if you are a user).  
3. If assisting, sign in with your **Microsoft account**.  
4. Provide the **security code** generated to the user for connection.  
5. The user enters the code to grant access.  
6. Use the **control or view-only options** depending on the session type.  
7. End the session once troubleshooting is complete.

---

## Using TeamViewer

1. Ensure TeamViewer is installed on both devices.  
2. Open TeamViewer and locate your **ID** and **Password**.  
3. Share the **ID** and **Password** with the IT support technician.  
4. Technician enters the credentials and requests control.  
5. Confirm access on your workstation if prompted.  
6. End the session when troubleshooting is complete.  

---

## Common Issues & Troubleshooting

| Issue | Possible Cause | Solution |
|-------|----------------|---------|
| **Cannot connect via RDP** | Network or firewall restrictions | Verify VPN connection and check firewall settings. |
| **Quick Assist code not accepted** | Code expired or entered incorrectly | Generate a new code and ensure the user enters it correctly. |
| **TeamViewer cannot connect** | Internet or version mismatch | Ensure both devices are online and updated to the same TeamViewer version. |
| **Remote session laggy or slow** | Poor network connection | Check network speed and reduce bandwidth usage on local applications. |
